Hola Mundo

Este es un ejemplo de un documento HTML básico que muestra un mensaje de "Hola Mundo". Puedes abrir este archivo en un navegador web para ver el resultado.

master: tiene el código productivo develop: tiene el código en desarrollo rama_propia: es una rama creada para trabajar en una nueva funcionalidad o corrección de errores sin afectar el código principal en develop o master. Una vez que los cambios en rama_propia estén listos y probados, se pueden fusionar (merge) de vuelta a develop para su integración y posterior despliegue a producción.
